Dots and Boxes: High level ordered list

  1. Make a square grid of dots that is 15 x 15
  2. Access user input
    1. Which two dots they draw their line between
    2. Completed a box
    3. Label/mark it belonging to the players
  3. Validate user move
  4. Evaluate possible moves
    1. Where there is no line drawn
  5. Choose computer moves
    1. Update board each turn
    2. Where lines are located
    3. If there are completed boxes who won them
  6. Test win condition
    1. No more moves
    2. Count number of boxes each player completed
    3. Player with most boxes wins
